# Cricket Analytics Frontend

## Overview
React frontend for my Cricket Analytics Final Year Project.
Built with React 18, Tailwind CSS, and Framer Motion for smooth animations.

## Features
- Clean, modern UI design
- 7-stage data pipeline interface
- Real-time status updates
- Responsive design for presentations

## Setup
1. Install dependencies: `npm install`
2. Run development server: `npm start`
3. App will be available at: http://localhost:3000

## Project Structure
- src/App.js - Main application component
- src/components/ - Reusable UI components
- src/pages/ - Individual module pages
- src/index.css - Global styles

## Pages
1. Data Collection - Web scraping interface
2. Data Cleaning - Data preprocessing
3. Data Transformation - Feature engineering
4. EDA - Data visualization
5. Modeling - ML predictions
6. Evaluation - Model performance
7. Export - Results export
